    Dale! is a proposal to help kids to learn to read and write. They develop in company with Globant Labs a web app to help kids in a fun and easy way through different games.

    Role

    Visual Designer


    Deliverables

    UI Kit
    UI Design


    Team

    Labs Globant


    Year

    2020

    Challenge

    After a great response to the adoption of the Web App, Labs decided to start working on the mobile app for Android users. The decision to start with this part of the market was because Dale! is conceived for the persona that was created as a principal consumer of the app.

    For a quick release, the developers reuse the library that was already constructed for the Web App.

    Goal

    The principal job of this stage of the project was to create the best experience for the personas and that it becomes easy to use for kids to give them a fun and educative app.

    So after creating some adaptation of the screens of the Web App for the mobile version, I start looking for ways to improve the look and feel through create a UI kit and paid attention to the interactions that occur in the mobile app.

    Next Steps

    Start conducting usability testing to find the pain points of users and create a list with the correct components of the app. In this way becomes easier to create a useful UI library.
